# 하루기록

식사, 음주, 컨디션과 짧은 메모를 날짜별로 남기는 개인 생활 기록 달력입니다.

## 기능

- 아침·점심·저녁·간식을 쉼표로 나눈 음식 항목으로 기록
- 음주 여부와 종류·양·메모 기록
- 5단계 컨디션과 하루 메모 기록
- 월간 달력의 기록 표시와 주간·월간 식단 통계
- 입력 내용 자동 저장 및 날짜별 초기화
- 전체 기록 JSON 내보내기·불러오기

기록은 공용 보관함 계층을 통해 현재 브라우저의 `IndexedDB`에 자동 저장되며, 도구함에서 파일 보관함을 연결하면 그 파일에도 백업할 수 있습니다. `IndexedDB`를 사용할 수 없는 환경에서는 기존 `localStorage`로 자동 전환합니다. 계정이나 외부 서버로 전송되지는 않으므로, 브라우저 데이터를 지우거나 다른 기기로 옮기기 전에는 연결 보관함 또는 JSON 파일을 보관하세요.

## 실행

정적 페이지이므로 `index.html`을 열거나 저장소 루트에서 간단한 로컬 서버를 실행하면 됩니다.

```bash
python3 -m http.server 8000
```

이후 `http://localhost:8000/daily-log/`에 접속합니다.

## 테스트

node --test tests/*.test.js
