# 꾸준표

매일의 습관을 체크하거나 수량으로 기록하고, 월간 달력과 달성률로 흐름을 확인하는 브라우저 기반 습관 추적기입니다.

## 기능

- 여러 습관 생성·수정·삭제
- 완료 체크형과 목표 수량형 기록
- 월요일부터 일요일까지 반복 요일 설정
- 목표 대비 부분 달성률과 월간 히트맵
- 현재·최장 연속 기록과 월간 통계
- 브라우저 `IndexedDB` 자동 저장과 연결 파일 보관함 백업
- JSON 백업·복원과 월간 PNG 저장
- 외부 서버 없이 브라우저에서만 데이터 처리

## 실행

빌드나 패키지 설치 없이 `index.html`을 열면 됩니다. 로컬 서버가 필요하면 저장소 루트에서 실행합니다.

```bash
python3 -m http.server 8000
```

테스트는 Node.js 18 이상에서 실행합니다.

```bash
node --test tests/habit.test.js
```

## 데이터 보관

기록은 공용 보관함 계층을 통해 현재 브라우저의 `IndexedDB`에 저장되며, 도구함에서 파일 보관함을 연결하면 그 파일에도 백업할 수 있습니다. `IndexedDB`를 사용할 수 없는 환경에서는 기존 `localStorage`로 자동 전환합니다. 서버로는 전송되지 않으므로 브라우저 데이터를 지우거나 다른 기기로 이동하기 전에는 연결 보관함 또는 화면의 **데이터 내보내기**로 백업을 보관하세요.
